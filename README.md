# Numerical Computation with Python

This repository covers fundamental topics in **numerical methods** implemented in Python, with a focus on solving mathematical problems arising in science and engineering.

---

##  Objective

The goal of this repository is to provide practical implementations of numerical methods, enabling the solution of real-world problems using Python.

---

### 🔹 Nonlinear Equations
- Root finding for single-variable equations:
  - Bisection method  
  - Newton's method  
- Newton's method for systems of nonlinear equations  
 

### 🔹 Approximation and Interpolation
- Polynomial interpolation  
- Function approximation techniques  

### 🔹 Numerical Calculus
- Numerical differentiation  
- Numerical integration  

### 🔹 Ordinary Differential Equations (ODEs)
- Initial value problems:
  - Taylor methods  
  - Runge-Kutta methods  
  - Implicit methods  
  - Predictor-corrector methods
